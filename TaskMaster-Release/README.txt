TaskMasterA modern, portable, and easy-to-use task management application. It doesn't require Python installation or technical knowledge.How to UseDownload and run TaskMaster.exe.No installation is required.The database is saved locally in the same folder.That's it! A new application window will open where you can create, organize, and filter your tasks. You can also create folders, change the language (English/Spanish), and everything saves automatically.RequirementsWindows 10/11 (with Microsoft Edge installed; most PCs already have it).No internet connection is required to function.How to UninstallSimply delete the folder containing the executable and the database (todos.db).How to ShareYou can copy the TaskMaster.exe file and the templates folder to any other Windows PC.Enjoy organizing your tasks with TaskMaster!--------------------------------------------------------------------------------------------------------------TaskMasterEs una aplicación de gestión de tareas moderna, portable y fácil de usar. No requiere instalación de Python ni conocimientos técnicos.¿Cómo usar?Descarga y ejecuta `TaskMaster.exe`No requiere instalación.La base de datos se guarda localmente en la misma carpeta.¡Listo!Se abrirá una ventana de la aplicación donde podrás crear, organizar y filtrar tus tareas.Puedes crear carpetas, cambiar el idioma (inglés/español), y todo se guarda automáticamente.RequisitosWindows 10/11 (con Microsoft Edge instalado, la mayoría de PCs ya lo tienen).No requiere conexión a internet para funcionar.¿Cómo desinstalar?Simplemente borra la carpeta donde está el ejecutable y la base de datos (`todos.db`).¿Cómo compartir?Puedes copiar el archivo `TaskMaster.exe` y la carpeta `templates` a cualquier otra PC con Windows.¡Disfruta organizando tus tareas con TaskMaster!
